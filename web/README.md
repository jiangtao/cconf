# ccconfig Landing Page

This is the official landing page for ccconfig, built with Next.js 14 and Tailwind CSS.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

The landing page is automatically deployed to Vercel on push to main.

- Live: https://ccconfig.vercel.app
- Vercel Dashboard: https://vercel.com/jiangtao/ccconfig

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Vercel
