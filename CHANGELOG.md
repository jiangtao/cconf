# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Go CLI implementation for cross-platform support
- Internationalization (i18n) support for English and Chinese
- GitHub Actions CI/CD workflows
- Makefile for cross-platform builds
- Configuration file support (~/.ccconfig.yaml)
- Unit tests for core packages

## [1.0.0] - Bash Scripts

### Added
- Initial Bash script implementation
- backup.sh - Automatic backup script
- restore.sh - Restore configuration script
- cache-plugin.sh - Plugin cache management
- Git version control integration
- Automatic sensitive info removal (API Token)
- Project configuration collection
